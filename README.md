## Características

- Diseño responsive
- Galería de productos
- Modal de detalles
- Integración con WhatsApp
- Página de contacto
- Animaciones visuales
- Hosting en Microsoft Azure
- Dominio personalizado con HTTPS

## Tecnologías utilizadas

- Angular
- TypeScript
- HTML5
- CSS3
- Microsoft Azure
